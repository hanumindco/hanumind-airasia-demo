A personal AI assistant built into the AirAsia Ride driver app. Every driver gets an assistant that knows their earnings, compliance status, zones, and history.

Three-layer intelligence reduces API costs by up to 55%:

Layer 1 FAQ Cache

Common questions answered instantly. Zero API cost. Covers PSV, EVP, BUDI95, earnings, zones, incentives.

Layer 2 Response Cache

Repeat questions served from 24-hour cache. Zero API cost. 100 drivers ask same question Claude called once.

Layer 3 Personalised AI

Unique questions answered by Claude with driver's actual data earnings gap, target, tomorrow's peak zones, compliance status.
